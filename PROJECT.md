#personal-webby 

purpose: personal portfolio site showcasing product management and design work

tech stack:
- next.js
- tailwind css

goals: 
- clean, modern design
- easy to navigate
- mobile responsive
- fast loading
- easy to iterate and deploy 

style: editorial, warm neutral, strong typography, generous spacing 

Landing page sections: 
1) Hero/Intro: big "portfolio" title, version text, name, short intro lines, headshot image on left.
2) Work sections: repeating blocks with a title line (role / org / date) and rectangular section underneath. Work sections should be clickable. 

Constraints:
- One page only for now (no routing).
- Prioritize layout, typography, spacing, responsiveness.
